## 주요기능
- Component item 클릭 시 CSS 복사
- color picker를 이용한 색상 입히기
- Category에 따른 Element 등록
- Firebase를 이용한 등록 및 조회

## 기술적 의사결정
- Vite : 매번 프로젝트를 CRA로 진행했지만 빌드 시간이 오래걸렸다. Vite의 장점인 HMR을 이용해 빠른 빌드시간과 Vite를 이용한 프로젝트를 경험을해보고 싶었기 때문에 채택
